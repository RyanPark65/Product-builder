# 프로젝트: 저녁 메뉴 추천

## 개요

이 프로젝트는 사용자에게 저녁 식사 메뉴를 추천하는 고품질 웹 애플리케E이션을 만드는 것을 목표로 합니다. Google AdSense 승인 기준을 충족하기 위해, 사용자에게 명확한 가치를 제공하고, 신뢰할 수 있으며, 풍부한 콘텐츠를 갖춘 사이트를 구축합니다.

## 디자인 및 기능

- **전문적이고 신뢰감 있는 UI:** 깨끗하고 현대적인 디자인을 적용하여 사용자가 콘텐츠에 집중하고 사이트를 신뢰할 수 있도록 합니다.
- **향상된 사용자 경험(UX):** 직관적인 네비게이션, 빠른 로딩 속도, 시각적 피드백(로딩 스피너 등)을 통해 사용자 경험을 극대화합니다.
- **풍부한 콘텐츠:** 단순한 메뉴 이름뿐만 아니라, 관련 이미지와 레시피 검색 링크를 함께 제공하여 사용자에게 더 많은 정보를 제공합니다.
- **반응형 레이아웃:** 모바일과 데스크톱 등 다양한 화면 크기에서 최적의 경험을 제공합니다.
- **명확한 사이트 구조:** '홈', '소개', '문의' 등 명확한 페이지 구성을 통해 사용자가 쉽게 사이트를 탐색할 수 있도록 합니다.

## AdSense 승인을 위한 개선 계획

1.  **`index.html` 구조 개편:**
    -   의미 있는 `<header>`, `<main>`, `<footer>` 태그를 사용하여 시맨틱한 구조로 변경합니다.
    -   헤더에는 '홈', '소개', '문의' 페이지로 이동할 수 있는 네비게이션 메뉴를 추가합니다.
    -   푸터에는 저작권 정보와 소셜 미디어 링크(플레이스홀더)를 추가하여 신뢰도를 높입니다.
    -   메뉴 추천 영역을 시각적으로 더 매력적인 카드로 변경합니다.

2.  **`about.html` 페이지 신규 생성:**
    -   사이트의 목적과 기능을 설명하는 '소개' 페이지를 만들어 사이트의 전문성과 신뢰도를 높입니다.

3.  **`style.css` 스타일 개선:**
    -   사이트 전반의 디자인을 더 세련되고 전문적으로 다듬습니다.
    -   헤더, 푸터, 네비게이션, 메뉴 카드 등 모든 요소에 일관된 스타일을 적용합니다.
    -   모바일 환경에서의 사용성을 고려하여 반응형 디자인을 강화합니다.

4.  **`main.js` 기능 향상:**
    -   단순 텍스트가 아닌, 이미지와 레시피 검색 링크가 포함된 정보성 카드를 동적으로 생성합니다.
    -   '다른 메뉴 추천받기' 버튼 클릭 시, 로딩 스피너를 표시하여 사용자에게 시각적 피드백을 제공합니다.
    -   `menu-database.js`의 방대한 메뉴 목록을 활용하되, 각 메뉴에 대한 임시 이미지를 동적으로 생성하여 보여줍니다.

5.  **`contact.html` 페이지 개선:**
    -   개선된 전체 디자인과 일관성을 유지하도록 스타일을 수정합니다.

6.  **AdSense 코드 최적화:**
    -   사용자 경험을 해치지 않는 위치에 광고 단위를 배치합니다.
